1. Requirement

  • Symfony >= v2.3

2. Installation

Use composer

{% highlight bash %} composer require a2lix/auto-form-bundle {% endhighlight %}

After the successful installation, add in your AppKernel.php file

{% highlight php %} // in AppKernel::registerBundles() $bundles = array( // ... new A2lix\AutoFormBundle\A2lixAutoFormBundle(), // ... ); {% endhighlight %}

3. Configuration

There is no minimal configuration. Full list:

{% highlight yaml %} # app/config/config.yml a2lix_auto_form: excluded_fields: [id, locale, translatable] # [1] {% endhighlight %}
